# Payment method icons (footer)

These 5 icons appear in one row in the footer (small, evenly spaced).

**Current files used by the footer:**

| Filename            | Label           |
|---------------------|-----------------|
| `visa-card.svg`     | Visa            |
| `mastercard-card.svg` | Mastercard    |
| `amex-card.svg`     | American Express|
| `paypal-card.svg`   | PayPal          |
| `applepay-card.svg` | Apple Pay       |

**Size:** Icons are displayed at 40px height. Replace any file with the same filename to update.
