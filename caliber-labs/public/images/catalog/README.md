# Product catalog card images

Place one PNG per product here. **Filename must be exactly `{slug}.png` in lowercase** (e.g. `tesamorelin.png`).  
These images appear in the product cards on the Products catalog page. Lowercase is required for deployment (e.g. Vercel/Linux).

| Filename | Product |
|----------|---------|
| `tesamorelin.png` | Tesamorelin |
| `sermorelin.png` | Sermorelin |
| `nad-plus.png` | NAD+ |
| `glow.png` | GLOW |
| `wolverine.png` | Wolverine |
| `bpc-157.png` | BPC-157 |
| `epitalon.png` | Epitalon |
| `ghk-cu.png` | GHK-CU |
| `cjc1295-ipamorelin.png` | CJC 1295/Ipamorelin |
| `glp-3r.png` | GLP-3R |
| `bremelanotide.png` | Bremelanotide |
| `mots-c.png` | MOTS-C |
| `bac-water.png` | BAC Water |

Use 4×5 aspect ratio (e.g. 400×500 px) to match the card layout. Add `nad-plus.png` if it’s missing.

**After cropping or replacing images:** Save over the existing file with the same filename. The app serves these directly, so a refresh (or hard refresh if needed) will show the new crop.
